Factorial Calculation
📌 Description

This project is a simple Python program that calculates the factorial of a given number.
It demonstrates the use of loops and basic mathematical operations.

🚀 Features
Calculates the factorial of a number
Accepts user input via terminal
Handles basic input validation

🛠️ Technologies
Python

▶️ How to Run
Clone the repository:
git clone https://github.com/LauroCalabria/Python-Factorial-Calculation.git

Navigate to the folder:
cd python-factorial-calculation

Run the file:
Cálculo do Fatorial.py
